формула: $ S = \frac{1}{(1-P)+\frac{P}{K}} $

S - теоретическое ускорение

Р - доля программы, которую можно распараллелить (от 0 до 1)

К - количество процессоров

1) `P=0.1`, `K=10_000`.
   $ S = \frac{1}{0.9+\frac{0.1}{10000}} = \frac{1}{0,90001} \approx 1,11 $
 т.е. программа ускорится на `11%`
2) $ A + \frac{B}{N} = 100 $, => $ N = \frac{250}{50} = 5 $. ответ: 5 процессоров.
3) $ 10 = \frac{1}{(1-P)+\frac{P}{100}} = \frac{100}{100 - 99P} $, =>
   $ P = \frac{90}{99} \approx 0,9090 \approx 0,91 $. ответ: надо, чтобы 91% программы было распараллелено.
